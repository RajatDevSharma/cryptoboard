# Cryptoboard
*vBeta*

A dashboard for checking the price of selected cryptocurrency.

Built with :
- [Streamlit](https://streamlit.io/)
- [CoinGecko API](https://www.coingecko.com/en/api)

### To Run

```commandline
streamlit run server.py
```
